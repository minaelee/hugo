Hi, I’m Minae.

I spun up this page using Hugo, a static site generator that’s written with the Go programming language. You don’t need to know Go to use it. I used a starter theme called Ananke, the recommended theme from Hugo’s quick start documentation.

I used GitHub Pages to deploy this Hugo single-page site, following Hugo’s instructions.

The complete list of technologies used

  • Hugo with Ananke theme
  • Chocolatey and Powershell to install Hugo on Windows
  • GitHub Pages
  • GitHub Actions
  • git CLI
  • Markdown
  • TOML and YAML(https://yaml.org/) data formats
  • Tachyons CSS library
  • Code editor (VS Code) and bash shell
