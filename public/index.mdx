# Hi, I'm Minae.

I spun up this page using [Hugo](https://gohugo.io/), a static site generator that's written with the Go programming language. You don't need to know Go to use it. I used a starter theme called [Ananke](https://github.com/theNewDynamic/gohugo-theme-ananke), the recommended theme from [Hugo's quick start documentation](https://gohugo.io/getting-started/quick-start/).

I used [GitHub Pages](https://pages.github.com/) to deploy this Hugo single-page site, following [Hugo's instructions](https://gohugo.io/hosting-and-deployment/hosting-on-github/).

## The complete list of technologies used

- [Hugo](https://gohugo.io/) with [Ananke theme](https://github.com/theNewDynamic/gohugo-theme-ananke)
- [Chocolatey](https://chocolatey.org/install) and [Powershell](https://learn.microsoft.com/en-us/powershell/) to install Hugo on Windows
- [GitHub Pages](https://pages.github.com/)
- [GitHub Actions](https://docs.github.com/en/actions)
- [git CLI](https://git-scm.com/book/en/v2/Getting-Started-The-Command-Line)
- [Markdown](https://en.wikipedia.org/wiki/Markdown)
- [TOML](https://toml.io/en/) and YAML(https://yaml.org/) data formats
- [Tachyons](https://tachyons.io/) CSS library
- Code editor ([VS Code](https://code.visualstudio.com)) and [bash shell](https://www.gnu.org/software/bash/manual/html_node/What-is-Bash_003f.html)
